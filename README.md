# 問題 2

## Node と Typescript を使って正整数 N を入力すると N の階乗を表示する関数を作成

### N の階乗：

- N の階乗は N!と表します
- たとえば N=4 のとき 4! = 4 x 3 x 2 x 1 = 24 となり
  N=5 のとき 5! = 5 x 4 x 3 x 2 x 1 = 120 となります

### 要件：

- 以下のような関数を定義すること
- ex) const function = (N) => N!
- 関数の引数と返り値の型を定義すること
- 再帰関数を使うこと
- 入力値を変更して関数を実行するごとに、正しく計算された N の階乗の結果が何かしらの方法で表示・確認できること
